# gui-template-toolkit

OpenClaw skill toolkit with reusable Python tkinter GUI templates extracted from 4+ tools via Ralph Loop.

## Contents

- `scripts/tkinter_boilerplate.py` - Standard tkinter app skeleton with menu, status bar, and theming
- `scripts/progress_dialog.py` - Reusable progress dialog with cancel support
- `widgets/common_widgets.py` - Common widget components (labeled frames, scrollable lists, etc.)

## Usage

```python
from scripts.tkinter_boilerplate import TkinterApp
from scripts.progress_dialog import ProgressDialog
```
